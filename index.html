设置jupyterlab开机自启动

Posted on 2020-12-14

设置jupyter lab开机自启动

windows的任务计划程序这一块调用vbs有问题，暂时不清楚，但是手动调用vbs是没问题的。先这样。

1.编写bat文件，如果有错误很有可能是conda没有加入环境变量，设置一下。

@echo off
cd C:\Users\rhwu\anaconda3\Scripts
start /B jupyter lab –no-browser

2.编写vbs文件：

On Error Resume Next
Dim oShell
Set oShell= WScript.CreateObject(“WScript.Shell”)
a = oShell.Run (“cmd /C C:\Jupyter.bat”,0)

3.双击vbs运行即可。
