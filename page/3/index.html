slice语句生成C#的list：

//dat解析输入值     
struct ResolInput
{
    string DatPath;                                //路径+iba文件名
    string hotNo;                                //热卷号
 }; 
 ["clr:generic:List"] sequence<ResolInput> ResolInputSeq;

slice2cpp 转换为C++语言后，sequence为vector，但是在C#中生成的seald class 类名加了后缀helper,变成了ResolInputSeqHelper。

想要在生成的cs文件中 将sequence生成为list 需添加：[“clr:generic:List”] 否则生成的是Array。Ice 3.7.1 Documentation p649

slice文件中定义结构体最稳定，字典和其他数据结构可能不稳定，字典、list等特殊的数据结构可以在各自的语言里进行定义实现。
