SQL server 更改数据库目录，并还原数据库：

1.分离该数据库，右键后勾选删除和更新；

2.将mdf和ldf拷贝到目标目录，并将文件权限完全放开；

3.数据库右键附加，确认是目标路径位置；

4.还原数据库。

如果数据库还原失败，一直显示正在还原，通过执行语句进行解决。

还原数据库时，注意数据库位置要更改一下。

首先将数据库的限制访问改成single：

右侧选择页中的文件进行mdf和log文件的位置设置。

之后在选项中勾选覆盖现有数据库
